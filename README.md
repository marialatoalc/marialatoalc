## 🛡️ Servidor Pi-hole con Docker

**Estado:** 🟢 Activo

Despliegue de un servidor **Pi-hole** mediante contenedores Docker para el filtrado de publicidad y la gestión centralizada de DNS en la red local.

### Funcionalidades
- Bloqueo de anuncios y dominios maliciosos
- Gestión DNS centralizada
- Despliegue mediante Docker Compose
- Administración sencilla y portable

### Resultados
- Reducción del tráfico publicitario en la red local
- Mejora del control sobre las consultas DNS
- Infraestructura fácilmente replicable y mantenible

### Tecnologías utilizadas
`Docker` `Docker Compose` `Linux` `DNS` `Pi-hole`

```bash
docker compose up -d
```

### Funcionalidades

- Supervisión del estado de contenedores Docker

- Integración con más servicios
